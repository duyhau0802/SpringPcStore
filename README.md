# Project's Summary
PC Store Project using Spring boot API and React, 
Design the database with an open architecture so that it can be expanded to e-commerce later.currently, it's only used for one store.

# Technologies
## FRONTEND
- React
- Vite
- Bootstrap
- Fetch API
## BACKEND
- Spring Boot API
- Spring Data JPA
- Hibernate
- MySQL
- CORS
